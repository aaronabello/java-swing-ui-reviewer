Study the syntax, memorize the patterns, type Java code in a Notepad-style editor, check common mistakes, and view a possible output preview.

Reviewer Notes

1. Universal pattern to remember

  • JFrame = main window
  • JPanel = container
  • JLabel = text label
  • JTextField = text box
  • JComboBox = dropdown list
  • JButton = button
  • JTextArea = large output area
  • GridLayout = easiest form layout
  • BorderLayout = title + center + button

Most UI-only Swing exam problems are just this pattern: create the frame, choose a layout, add labels and inputs, then show the window.

3. Universal basic skeleton

import javax.swing.*;
import java.awt.*;

class SampleUI extends JFrame
{
    JLabel firstLabel, secondLabel, thirdLabel, resultLabel;
    JTextField firstField, secondField, thirdField, resultField;
    JButton computeButton;

    public static void main(String[] args)
    {
        new SampleUI();
    }

    public SampleUI()
    {
        setTitle("Sample UI");
        setSize(500, 350);
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        setLocationRelativeTo(null);
        setLayout(new GridLayout(5, 2, 10, 10));

        firstLabel = new JLabel("First Input:");
        secondLabel = new JLabel("Second Input:");
        thirdLabel = new JLabel("Third Input:");
        resultLabel = new JLabel("Result:");

        firstField = new JTextField();
        secondField = new JTextField();
        thirdField = new JTextField();
        resultField = new JTextField();

        computeButton = new JButton("Compute");

        add(firstLabel);
        add(firstField);
        add(secondLabel);
        add(secondField);
        add(thirdLabel);
        add(thirdField);
        add(resultLabel);
        add(resultField);
        add(new JLabel(""));
        add(computeButton);

        setVisible(true);
    }
}

6. Template for a cleaner modern-looking UI

Use a title label at the top, a form panel in the center, and a button panel at the bottom.

JPanel mainPanel = new JPanel(new BorderLayout(10, 10));
mainPanel.setBorder(BorderFactory.createEmptyBorder(20, 20, 20, 20));

JLabel titleLabel = new JLabel("EXAM USER INTERFACE", JLabel.CENTER);
titleLabel.setFont(new Font("Arial", Font.BOLD, 20));

JPanel formPanel = new JPanel(new GridLayout(4, 2, 10, 10));
JPanel buttonPanel = new JPanel();
buttonPanel.add(new JButton("Submit"));

mainPanel.add(titleLabel, BorderLayout.NORTH);
mainPanel.add(formPanel, BorderLayout.CENTER);
mainPanel.add(buttonPanel, BorderLayout.SOUTH);

add(mainPanel);

7. Template with TextArea output

Use this when the problem says output, summary, receipt, report, or multi-line result.

JTextArea outputArea = new JTextArea();
outputArea.setEditable(false);
JScrollPane scrollPane = new JScrollPane(outputArea);

add(scrollPane, BorderLayout.CENTER);

Important memory line: large output area usually needs JScrollPane.

11. Layout cheat sheet

  • GridLayout = best for forms
  • BorderLayout = title on top, form in center, button below
  • FlowLayout = best for button panels
setLayout(new GridLayout(4, 2, 10, 10));
setLayout(new BorderLayout(10, 10));
JPanel buttonPanel = new JPanel();
buttonPanel.setLayout(new FlowLayout());

12. Quick formula for counting GridLayout rows

  • 3 inputs + result + button → GridLayout(5, 2, 10, 10)
  • 2 inputs + result + button → GridLayout(4, 2, 10, 10)
  • 4 labels and 4 fields → 4 rows, 2 columns inside a form panel

13. How to translate the question fast during exam

  • 3 labels + 3 text fields + result + button → pure text-field form
  • 3 labels + dropdown list → replace JTextField with JComboBox
  • Student registration form → text fields + combo box + possible radio buttons
  • Reservation / order / billing / calculator → labels + fields + button
  • Receipt or report output → use JTextArea with JScrollPane

14. Plug-and-play exam template

import javax.swing.*;
import java.awt.*;

class ExamUI extends JFrame
{
    JLabel label1, label2, label3, label4;
    JTextField field1, field2, field3, field4;
    JButton button1;

    public static void main(String[] args)
    {
        new ExamUI();
    }

    public ExamUI()
    {
        setTitle("Exam UI");
        setSize(500, 350);
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        setLocationRelativeTo(null);

        JPanel mainPanel = new JPanel(new BorderLayout(10, 10));
        mainPanel.setBorder(BorderFactory.createEmptyBorder(20, 20, 20, 20));

        JPanel formPanel = new JPanel(new GridLayout(4, 2, 10, 10));

        label1 = new JLabel("Label 1:");
        label2 = new JLabel("Label 2:");
        label3 = new JLabel("Label 3:");
        label4 = new JLabel("Result:");

        field1 = new JTextField();
        field2 = new JTextField();
        field3 = new JTextField();
        field4 = new JTextField();

        formPanel.add(label1);
        formPanel.add(field1);
        formPanel.add(label2);
        formPanel.add(field2);
        formPanel.add(label3);
        formPanel.add(field3);
        formPanel.add(label4);
        formPanel.add(field4);

        JPanel buttonPanel = new JPanel();
        button1 = new JButton("Compute");
        buttonPanel.add(button1);

        mainPanel.add(formPanel, BorderLayout.CENTER);
        mainPanel.add(buttonPanel, BorderLayout.SOUTH);

        add(mainPanel);
        setVisible(true);
    }
}

19. One-line reminders

JFrame = window JPanel = section/container JLabel = text JTextField = single-line input JTextArea = multi-line input/output JButton = button JComboBox = dropdown JRadioButton = one choice JCheckBox = many choices GridLayout = form table BorderLayout = top / center / bottom layout

20. What not to add in a UI-only exam

  • Do not add implements ActionListener
  • Do not add addActionListener(this)
  • Do not add actionPerformed()
  • Only show the user interface unless the teacher asks for events

21. Notepad compile and run

File name must match class name.

Example:
class StudentForm extends JFrame
Save as: StudentForm.java

Compile:
javac StudentForm.java

Run:
java StudentForm

22. Exam survival tips

  • If you panic, use only JFrame, JLabel, JTextField, JButton, JComboBox, GridLayout.
  • If the result should not be typed, use resultField.setEditable(false);
  • If you want cleaner output, use a title label on top, form panel in center, button panel below.
  • During the exam, usually you only change the class name, title, labels, dropdown choices, and button text.

23. Memorization drill

  • What import is needed for Swing? import javax.swing.*;
  • What import is needed for layouts, fonts, and colors? import java.awt.*;
  • What word makes the class a frame? extends JFrame
  • What shows the frame? setVisible(true);
  • What centers the window? setLocationRelativeTo(null);
  • What layout is easiest for forms? GridLayout
